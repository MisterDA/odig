Module Async_unix__.Async_sys

This module overrides everything in the Sys module that might block. Functions do the same thing as their counterparts in Sys, but instead return deferreds. For a description of their semantics see the documentation for the Sys module.

val when_file_exists : ?⁠follow_symlinks:bool -> ?⁠poll_delay:Core.Time.Span.t -> string -> unit Async_unix__.Import.Deferred.t

when_file_exists ?poll_delay file returns a deferred that becomes determined when file exists. The default poll delay is 0.5 seconds. It raises an exception if it can not check whether the file is there, in the same cases file_exists returns `Unknown.

val when_file_changes : ?⁠poll_delay:Core.Time.Span.t -> string -> Core.Time.t Async_unix__.Import.Pipe.Reader.t

when_file_changes file polls file using stat and writes file's mtime to the pipe every time it changes. The first time in the pipe will be file's current mtime. To stop polling, close the pipe.
