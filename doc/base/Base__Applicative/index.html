The following functors give a sense of what Applicatives one can define.

Of these, Of_monad is likely the most useful. The others are mostly didactic.

module Of_monad : functor (M : Base__.Monad.S) -> S with type 'a Of_monad.t := 'a M.t

Every monad is Applicative via:

module Compose : functor (F : S) -> functor (G : S) -> S with type 'a Compose.t = 'a F.t G.t
module Pair : functor (F : S) -> functor (G : S) -> S with type 'a Pair.t = 'a F.t * 'a G.t
module Const : functor (Monoid : sig ... end) -> S with type 'a Const.t = Monoid.t

Every monoid gives rise to a constant Applicative.
