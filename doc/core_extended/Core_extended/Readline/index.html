Module Core_extended.Readline

Interactive line editing.

This implements very basic readline abilities: backspace, left and right arrows work as expected. There's also a history that can be browsed through the up and down arrows.

the string to use as a prompt (default value ">")

parameter history

the mutable value used as a history. The deault value is History.default. If you don't want any history ou should use History.null

the function use to complete on tab. By default there is no completion.
