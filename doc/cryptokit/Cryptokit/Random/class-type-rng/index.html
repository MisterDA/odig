Class type Random.rng

Generic interface for a random number generator.

method random_bytes : bytes -> int -> int -> unit

random_bytes buf pos len stores len random bytes in byte array buf, starting at position pos.

method wipe : unit

random_bytes buf pos len stores len random bytes in byte array buf, starting at position pos.

Erases the internal state of the generator. Do not call random_bytes after calling wipe.
