Module Int.Infix

Infix operations.

val (-->) : 'a t -> key -> 'a

t --> k returns the current binding of k in t, or raises Not_found. Strictly equivalent to find k t.

val (<--) : 'a t -> (key * 'a) -> 'a t

t <-- (k, v) adds to t a binding from k to v and returns the result. Strictly equivalent to add k v Hamt.
