Branches

module type S = sig ... end

The signature for branches. Irmin branches are similar to Git branches: they are used to associated user-defined names to head commits. Branches havve a default value: the master branch.

module String : S with type t = string

String is an implementation of S where branches are strings. The master branch is "master". Valid branch names contain only alpha-numeric characters, -, _, ., and /.

module type STORE = sig ... end

STORE specifies the signature for branch stores.
