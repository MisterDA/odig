returns

the Toeplitz matrix associated with v. The constant diagonals are read from left to right from v.

raises Invalid_argument

if the length of v is not an odd number.

val random : ?⁠rnd_state:Stdlib.Random.State.t -> ?⁠from:float -> ?⁠range:float -> int -> int -> Lacaml_float64.mat

random ?rnd_state ?from ?range m n

returns

an mxn matrix initialized with random elements sampled uniformly from range starting at from. A random state rnd_state can be passed.

default = Random.get_state ()

parameter from

default = -1.0

parameter range

default = 2.0

val abs : Lacaml_float64.Types.Mat.unop

abs ?m ?n ?br ?bc ?b ?ar ?ac a computes the absolute value of the elements in the m by n sub-matrix of the matrix a starting in row ar and column ac. If b is given, the result will be stored in there using offsets br and bc, otherwise a fresh matrix will be used. The resulting matrix is returned.

parameter m

default = greater n s.t. ar + m - 1 <= dim1 a

parameter n

default = greater n s.t. ac + n - 1 <= dim2 a

parameter br

default = 1

parameter bc

default = 1

parameter b

default = fresh matrix with br + m - 1 rows and bc + n - 1 columns

parameter ar

default = 1

parameter ac

default = 1

val atan2 : Lacaml_float64.Types.Mat.binop

atan2 ?m ?n ?cr ?cc ?c ?ar ?ac a ?br ?bc b computes atan2(a, b) for the m by n sub-matrix of the matrix a starting in row ar and column ac with the corresponding sub-matrix of the matrix b starting in row br and column bc. If c is given, the result will be stored in there starting in row cr and column cc, otherwise a fresh matrix will be used. The resulting matrix is returned.

NOTE: WARNING! From a geometric point of view, the atan2 function takes the y-coordinate in a and the x-coordinate in b. This confusion is a sad consequence of the C99-standard reversing the argument order for atan2 for no good reason.

default = 1

parameter cc

default = 1

parameter c

default = fresh matrix with cr + m - 1 rows and cc + n - 1 columns

parameter br

default = 1

parameter bc

default = 1

parameter ar

default = 1

parameter ac

default = 1
