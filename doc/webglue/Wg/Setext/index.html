Module Wg.Setext

S-expressions as textual data.

Setext outputs list of atoms as strings separated by a single space character. The space between two atoms is suppressed if there's an "\@" atom between them. To output a single "\@" use "\@\@".

val output : ?⁠out_string:(Out.output -> string -> unit) -> ?⁠until_list:bool -> Out.output -> se list -> se list

output out_string until_list o sl outputs the sl as text. out_string is used to output the atoms (defaults to Out.s). If until_list is true (defaults to false) the function stops on the first s-expression list and returns the remaining s-expressions.

val output_lines : ?⁠out_string:(Out.output -> string -> unit) -> Out.output -> se list -> unit

output_lines out_string o sl outputs each list in sl with output.

val to_atom : se -> atom

to_atom se is se if se is an atom or the atom resulting from interpreting the list as @-text (the dictonary of the atom is the dictionary of the list).

val to_string : se list -> string

to_string sl is sl interpreted as text.
