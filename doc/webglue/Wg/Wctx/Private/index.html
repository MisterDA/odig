Module Wctx.Private

Private functions.

Private functions are for systems using webglue, they must not be used by formats.

val create : ?⁠locale:Wlocale.t -> ?⁠conf:atom -> (t -> id -> wmap option) -> id -> t

create locale conf db id is an evaluation context such that :

  • w.id and w.self binds to id.
  • w.locale binds to locale.
  • w.conf binds to conf.
  • db is used for map lookup.
