Module Wg.Wformat

Map formats and database.

A map format is a module implementing the signature T. It must be defined by calling define.

Format definition

type key_info = [
| `Required
| `Optional of se list option
| `Derived
]

The type for key information. `Required means that the key must be defined in the map. `Optional means that the key may be defined in the map and if a value is given it is used to defined the key when it is absent. `Derived means that the key is computed by the format at creation time.

type man_block = [
| `S of string
| `P of string
| `Pre of string
| `I of string * string
| `Noblank
| `Blocks of man_block list
]

The type for man page blocks.
