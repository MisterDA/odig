Module Wg.Wlocale

Locale identifiers and locale ranges.

Locale identifiers and ranges are BCP 47 language tags and ranges.

Locale identifier and ranges

type t = atom

The type for locale identifiers.

type range = atom

The type for locale ranges.

val is_locale : atom -> bool

is_locale a is true if a is a BCP 47 extended language range without wildcard subtags.

val is_range : atom -> bool

is_range a is true if a is a BCP 47 extended language range.

val subtags : atom -> string list

subtags s is the lowercased list of subtags of the locale or range s.

val compare : atom -> atom -> int

compare r r' orders r and r' by the number of subtags and, if equal, by by lexicographic order on the subtags.

val equal : atom -> atom -> bool

equal r r' is true iff the atoms r and r' are equal.

val matches : t -> range -> bool

matches l r is true if locale l matches range r.

Parsers

val p_locale : t Se.parser

p_locale parses an atom as a BCP 47 language tag.

val p_locale_range : range Se.parser

p_locale_range parses an atom as a BCP 47 language range tag.
